# Text

This component displays a text with any kind of html tag. By default it is a `span` element.

<Canvas of={TextStories.Default} />

> The `Text` component does not have default style. You could use the following
> design token for the font family - `fonts.normal` or `fonts.mono` - and the
> array `fontSizes` for the font size.

# Comparison with JupyterLab

> You can display the JupyterLab equivalent component through the toolbar.

In JupyterLab, you can use the font family CSS properties:
- `--jp-content-font-family`
- `--jp-ui-font-family`
- `--jp-code-font-family`

And for the font size:
- `--jp-content-font-size#` (range from 0 to 5)
- `--jp-ui-font-size#` (range from 0 to 3)
- `--jp-code-font-size` (unique value)

```jsx
<span
  style={{
    fontFamily: 'var(--jp-content-font-family)',
    fontSize: 'var(--jp-content-font-size1)',
  }}
>
  {'Text'}
</span>
```

> ⚠️ JupyterLab font size are using relative unit [em](https://developer.mozilla.org/en-US/docs/Web/CSS/length#relative_length_units_based_on_font)
> except for `--jp-content-font-size1`. This may have poor side effect like the following nested:

```jsx
<p
  style={{
    fontFamily: 'var(--jp-content-font-family)',
    fontSize: 'var(--jp-content-font-size1)',
  }}
>
  <span style={{fontSize: 'var(--jp-content-font-size3)'}}>
    {'Expected change'}
    <span style={{fontSize: 'var(--jp-content-font-size3)'}}>
      {'Unexpected'}
    </span>
  </span>
</p>
```
