# 🪐 🦕 Docusaurus

We maintain a [plugin](https://docusaurus.io/docs/next/using-plugins) for [Docusaurus](https://docusaurus.io) in the [docusaurus-plugin](https://github.com/datalayer/jupyter-ui/tree/main/packages/docusaurus-plugin) package folder. This allows you to embed Jupyter into any Docusaurus website.

An example with [Docusaurus](https://docusaurus.io) is available in this [repository folder](https://github.com/datalayer/jupyter-ui/tree/main/examples/docusaurus) and you can try it [online on this page](/docs/demos/cell).
