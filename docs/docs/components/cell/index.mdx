## With in-browser kernel

You can use an in-browser kernel by setting the `lite` property
of `Jupyter` component:

<StorybookView 
    story='components-cell--lite-python'
/>

```jsx
<Jupyter
    lite={true}
>
    <Cell
        source={`import sys
print(f"{sys.platform=}")

#...`}
    />
</Jupyter>
```
