# Comparison with JupyterLab

> You can display the JupyterLab equivalent component through the toolbar.

The JupyterLab button are the one visible in the dialogs. Their CSS rules
is spread between 3 CSS stylesheets (in addition to the theme):
- `@jupyterlab/application/style/buttons.css`
- `@jupyterlab/ui-components/style/styling.css`
- `@jupyterlab/apputils/style/dialog.css`

And they required to add manually the class `jp-mod-styled`. Otherwise,
the button will not be styled; here is an example of the primary button
missing that class:

<div dangerouslySetInnerHTML={{__html: createLabButton({label: 'Unstyled', variant: 'primary', noModStyled: true })}}></div>
